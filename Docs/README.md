## Copying Docs to Your Desktop

You can easily copy this entire documentation folder to your Desktop for offline reference:

### Option 1: Use the Python Script
Run the provided script from the repository root:
```bash
python copy_docs_to_desktop.py
```

### Option 2: Manual Copy
1. Navigate to the repository folder
2. Copy the entire `Docs` folder
3. Paste it to your Desktop

The documentation files are in Markdown format (.md) and can be read in any text editor or Markdown viewer.

## Reading Markdown Files

- **VS Code**: Open the .md files and press `Ctrl+Shift+V` (or `Cmd+Shift+V` on Mac) for preview
- **GitHub**: If viewing online, GitHub automatically renders Markdown
- **Text Editor**: Any text editor can open .md files as plain text
