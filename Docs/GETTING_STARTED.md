# Getting Started with Python Experiments

Welcome to the Python Experiments repository! This guide will help you get started with using the Python experiments included in this project.

## Prerequisites

- Python 3.x installed on your system
- A code editor (VS Code recommended)
- Basic understanding of Python programming

## Quick Start

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/cmotjsappliance/python-experiments.git
   cd python-experiments
   ```

2. **Open in Your IDE**
   - Open the project folder in VS Code or your preferred IDE
   - Navigate to the Python file you want to run

3. **Run the Experiments**
   - Open a Python REPL using:
     - Windows/Linux: `Ctrl + Shift + \``
     - Mac: `Cmd + Shift + \``
   - Save your code with `Ctrl + S` (Windows/Linux) or `Cmd + S` (Mac)
   - Run the code in the REPL
